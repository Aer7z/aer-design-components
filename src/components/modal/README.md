# API

## `<Modal>` Props说明

| 参数名 |       描述       |   类型   | 默认值 |
| :----: | :--------------: | :------: | :----: |
| title  | 模态对话框的标题 | `string` |   -    |
|  body  | 模态对话框的内容 | `string` |   -    |

## `<Modal>` Event说明

| 事件名 | 描述                 | 参数             |
| ------ | -------------------- | ---------------- |
| opened | 模态对话框打开时触发 | ev: `MouseEvent` |
| closed | 模态对话框关闭时触发 | ev: `MouseEvent` |
